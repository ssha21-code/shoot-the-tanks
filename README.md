# shoot-the-tanks
* This is a program made in Raylib C++. It is a small, fun game made in C++.
* The Player is a tank, which can shoot with unlimited ammo.
* The Player's turret faces the mouse pointer at all times.
* The Player has 100HP.
* In each wave, eigth Enemy tanks spawn, and face the Player at all times and shoot at the player.
* Each Enemy tank can reload from a random time from 0.5 to 1.5 seconds, and also have unlimited ammo.
* Enemy tanks move for 150 pixels and stop to face the player and shoot.
* Each Enemy tank has 10HP.
* The Enemy tanks face the Player's center and update their rotation the moment the Player moves.
* Enemy tanks do damage from 1HP to 3HP randomly, with each collision with the Enemy tank's bullet and the player.
* The game also has a pause and play feature, where the game can pause and play with the pause button.
* The controls of the game are:
  - Move: Key W (Moves towards mouse pointer)
  - Shoot: Left Mouse Button or Key Space
* When the player shoots a bullet, when the bullet hits an enemy tank, it does 1HP to 7HP randomly to the enemy tank.
* When all the Enemy tanks in a wave are defeated, the next wave starts, and the Enemy tanks respawn with different stats.
* The Player's goal is to move around and dodge all Enemy tank bullets, to shoot bullets at Enemy tanks and destroy them, and to defeat as many waves as possible.
* The user can play as many games as they like, and the highest number of waves defeated in each game of shoot-the-tanks will show on the home screen.
